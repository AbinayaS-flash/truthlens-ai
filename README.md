Project Name:
  TruthLens AI – AI-Powered Fact Verification & Bias Detection System

Problem:
  In today's digital world, people are exposed to large amounts of information from news websites, blogs, and social media platforms. It is often difficult to determine whether a piece of information is accurate, trustworthy, or biased. The rapid spread of misinformation and fake news can mislead users and influence decisions, making fact verification a challenging task.

Solution:
  TruthLens AI is an intelligent fact-verification system that analyzes information from multiple online sources and generates evidence-based responses. The platform retrieves relevant sources, evaluates their credibility, performs contradiction and bias analysis, and provides users with a confidence score along with supporting references. This enables users to quickly verify claims and access reliable information from transparent sources.

Technology Stack:
  Frontend: React.js, Vite, Tailwind CSS, Axios, React Router DOM
  Backend: FastAPI, Python 
  AI & NLP: LangGraph, LangChain, Groq LLM, Sentence Transformers, TextBlob, NLTK
  Search Engine: Tavily Search API
  Authentication: Firebase Authentication
  Deployment: Vercel, Render
  Version Control: Git, GitHub
